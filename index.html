RGB COLOR GAME PROJECT

RGB Color Game Project is a web development project that tests your RGB Color recognizing skills. For people who are not aware of RGB, RGB is a color model system that mixes red, green and blue colors with one another to form any color. This color model system looks like this: rgb(0,255,0). The rgb in this system stands for red, green and blue; the three numbers inside represent the amount of red, green and blue colors. So in rgb(0,255,0), there is no red, a lot of green and no blue, which means this is a green color. These numbers start from 0 and they can go up to 255, but that's it.

My RGB Color Game Project creates a guessing game where such a color model system will be given, and after seeing that system you will have to select which of the colors does that system represent. You can play this game unlimited time just by pressing new colors.

PENALTY SHOOTOUT

Penalty ShootOut is a basic game development project where two players score against each other until one of them acquires a specified amount of goal

In this project, you can name both players, you can select the number of games you want to play and after finishing one game, you can just press the reset button and play this game for unlimited amount of times.
